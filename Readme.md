# PROJETO DA MATERIA SISTEMAS CLIENTE/SERVIDOR

#### Descrição

O Projeto consiste em um sistema de cadastro de Livros de uma livraria, onde pode, **cadastrar, alterar, deletar e consultar** os livros no sistema
